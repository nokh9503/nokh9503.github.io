OX퀴즈

풀이

  1. “O”, “X”를 입력한다.
  2. result(결과값)과 cnt(연속된 점수) 변수를 만들어 연속된 “O”의 개수를 고려하여 점수를 계산한다.
  3. “x”가 나올 시 cnt 값을 0으로 초기화하여 문제 조건에 만족하게 만든다.

T = int(input())    # 테스트 케이스

for i in range(T):
    
    n = input()     # 'O', 'X' 입력
    result = 0      # 결과값
    cnt = 0         # 연속된 점수

    for j in range(len(n)):

        # 연속된 'O'값을 고려하여 계산
        if n[j] == 'O':
            cnt += 1
            result += cnt

        # 'X'이면 cnt을 0으로 초기화
        else:
            cnt = 0
            
    print(result)
